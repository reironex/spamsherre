TURBO GUIDE

RETURN TO MAIN PANEL

Step 1 – Install Firefox

Go to the PlayStore or AppStore and download the Firefox browser.

Step 2 – Login Account

Log in to your Facebook dummy account using the Firefox browser.

Step 3 – Cookie Extension

Install the "Cookie-Editor" extension from the Firefox add-ons store.

Step 4 – Get AppState

Open Cookie-Editor, click "Export", and copy the JSON string (AppState).

Step 5 – Paste & Launch

Return to ShareBoost, paste the code into the AppState field, and start boosting.
